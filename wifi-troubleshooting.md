# 🌐 How to Troubleshoot Wi-Fi Connection Issues

## 📖 Overview
This guide helps resolve common Wi-Fi connectivity problems for users.

---

## 🛠️ Step 1: Basic Checks
- Make sure Airplane Mode is OFF
- Confirm Wi-Fi is turned ON
- Check if other devices are connected

---

## 🛠️ Step 2: Restart Devices
- Restart the computer
- Restart the router/modem (unplug for 30 seconds)

---

## 🛠️ Step 3: Forget and Reconnect Network
1. Go to Wi-Fi settings
2. Click on the network
3. Select **Forget**
4. Reconnect and re-enter password

---

## 🛠️ Step 4: Run Troubleshooter (Windows)
- Right-click Wi-Fi icon
- Click **Troubleshoot problems**
- Follow on-screen steps

---

## ⚠️ When to Escalate
- No networks detected at all
- Router shows no internet light
- Issue persists after all steps
