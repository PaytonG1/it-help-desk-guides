# 📧 How to Set Up an Email Account in Outlook

## 📖 Overview
This guide explains how to add an email account to Microsoft Outlook.

---

## 🛠️ Step 1: Open Outlook
- Launch Microsoft Outlook
- Click **File → Add Account**

---

## 🛠️ Step 2: Enter Email
- Type email address
- Click **Connect**

---

## 🛠️ Step 3: Enter Password
- Input email password
- Click **Sign In**

---

## 🛠️ Step 4: Server Settings (If Needed)
- IMAP (recommended for most users)
- SMTP for sending emails

---

## ⚠️ Common Issues
- Wrong password
- Account not synced
- Server settings incorrect

---

## 🛠️ Step 5: Test Email
- Send a test email
- Confirm inbox sync

---

## 🧠 Skills Demonstrated
- Email client configuration
- Account troubleshooting
- Basic network/email protocols awareness
