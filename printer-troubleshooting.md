# 🖨️ How to Fix Common Printer Issues

## 📖 Overview
This guide helps resolve basic printer problems like offline status or print failures.

---

## 🛠️ Step 1: Check Connections
- Ensure printer is powered ON
- Check USB or Wi-Fi connection
- Verify correct printer is selected

---

## 🛠️ Step 2: Clear Print Queue
1. Go to **Devices and Printers**
2. Select printer
3. Click **See what's printing**
4. Cancel all documents

---

## 🛠️ Step 3: Restart Print Spooler
1. Press `Windows + R`
2. Type `services.msc`
3. Find **Print Spooler**
4. Click **Restart**

---

## 🛠️ Step 4: Reinstall Printer
- Remove printer from system
- Reinstall using driver or Windows update

---

## ⚠️ When to Escalate
- Printer not detected at all
- Hardware errors (paper jams repeatedly)
- Driver installation failure

---

## 🧠 Skills Demonstrated
- Hardware troubleshooting
